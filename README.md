# WhyWhyWhy

计划做一个AI互动版“十万个为什么”

## 功能特点

- 用《十万个为什么》书本做RAG
- 十万个为什么的问题和回答也可以作为微调的训练集
- 提取问题，随机展示，用户选择提问
- AI加工后回复，用户追问
- AI更加详细的解答

## 进度
- 目前提取了3438个问答对
